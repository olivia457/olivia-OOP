3.2 Sintaxis

Modificar tamaño y color

Selector: indica que elemento HTML se aplica.

Propiedad: lo que se quiere cambiar (ej. color, font-size).

Valor: el ajuste especifico (ej. blue, 20px).

3.3 Selectores

"Arquitectura de las aplicaciones web"

Una arquitectura web define como se organizan los componentes de software y hardware que la aplicación funcione.

Componentes principales:

Cliente(Frontend):

-El navegador del usuario.

-Usa HTML, CSSS, JavaScript.

-Interactua con el servidor para pedir o enviar datos

Servidor(Backend):

-Procesa las solicitudes de los clientes.

-Ejecuta lógica de negocio.

-Se comunica con la base de datos.

-Lenguajes comunes: Python(Flask/Django), Java(Spring).

Base de datos:

-Almacena y organiza la información.

-Ejemplos: MySQL, PostgreSQL, MongoDB.
